======================================== how to inactivate click event for window in js =========================================== https://fontawesome.com/ https://fonts.google.com/icons https://heroicons.com/ with: postcss 1. npm init 2. npm install -D tailwindcss postcss autoprefixer vite 3. npx tailwind init -p (this will will generate tailwind.config.js file) 4. put '*' in the tailwind.config.js file inside its content property's value "content:[]" - /** @type {import('tailwindcss').Config} */ module.exports = { content: ['*'], theme: { extend: {}, }, plugins: [], } 5. create a base folder -name it 'css' and create a css file inside it. then copy 3 lines codes mentioned below in the file - @tailwind base; @tailwind components; @tailwind utilities; 6. install extension 'Tailwind CDD IntelliSense' from vscode 7. Modify script section of package.json file as below "scripts": { "start": "vite", "build":"tailwindcss -i input.css -o ./css/output.css --minify", "watch":"tailwindcss -i input.css -o ./css/output.css --watch", "dev":"vite build" }, 8. In terminal run the tailwind with command: * for tailwind to run: npm run start * to build: tailwindcss -i input.css -o ./css/output.css --minify * keep it watch: tailwindcss -i input.css -o ./css/output.css --watch (it should be run in another terminal) * to pakage: vite build; * svg for hamburger menu
